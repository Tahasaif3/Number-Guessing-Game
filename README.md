## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Tahasaif3/Number-Guessing-Game.git
   ```
2. Navigate to the project directory:
   ```bash
   cd number-guessing-game
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
